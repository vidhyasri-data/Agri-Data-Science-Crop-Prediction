# Crop Prediction and Soil Health Analysis

### 🎯 Project Goal
To analyze soil health (NPK levels) and predict the best crops for farmers in Mayiladuthurai District.

### 🛠 Tools & Technologies
- **Programming:** Python (Scikit-Learn, Pandas)
- **Model:** Random Forest Classifier
- **Visualization:** Power BI & Tableau
- **Analysis:** SQL

### 📊 Results
- Achieved high accuracy in predicting crops based on soil conditions.
- Created interactive dashboards to visualize nutrient trends across the region.
